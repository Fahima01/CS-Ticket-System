1. What is JSX, and why is it used?
ans: JSX (JavaScript XML) used in React that allows to write HTML-like code inside JavaScript. It makes UI code easier to read and write.

2. What is the difference between State and Props?
ans: Props are used to pass data from a parent component to a child components. State is managed inside a component and we can change them. We can Update state component to re-use.

3. What is the useState hook, and how does it work?
ans: useState is a React hook used to add state to functional components. It returns a state variable and a function to update it. When the state changes, React re-renders the component.

4. How can you share state between components in React?
ans: State can be shared by lifting it up to a common parent component. The parent stores the state and passes it to child components through props. 

5. How is event handling done in React?
ans: Event handling event attributes like onClick, onChange, or onSubmit. A function is passed as the event handler. When the event occurs, the function is executed.
